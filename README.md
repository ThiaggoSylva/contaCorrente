# 💳 Conta Corrente

Aplicação de console desenvolvida em **C#** para simular operações básicas de uma **conta corrente**.

O programa permite realizar movimentações bancárias simples, como:

- saque
- depósito
- transferência
- consulta de saldo

---

## 📋 Descrição

Este projeto foi criado com o objetivo de praticar conceitos fundamentais de programação orientada a objetos em C#, trabalhando com:

- criação de classes
- atributos e métodos
- interação via console
- manipulação de saldo bancário
- transferência entre contas
- organização básica de código em arquivos separados

---

## ⚙️ Funcionalidades

✅ Realizar saque  
✅ Realizar depósito  
✅ Transferir valores entre contas  
✅ Consultar saldo da conta  
✅ Menu interativo no console
